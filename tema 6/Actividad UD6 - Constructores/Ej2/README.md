1.Crear una clase llamada Empleado con los siguientes atributos privados:
 • nombre (String)
 • salario (double)
 • horasSemanales (int)
2.Crear un constructor que obligue a indicar todos los atributos en el momento de crear el objeto. Validar dentro del constructor que:
 • el salario sea mayor o igual que 0
 • las horasSemanales estén entre 1 y 40
 • Si alguno de los valores no es válido, debe asignarse automáticamente el valor 0.
3.Crear una segunda clase llamada Ej02 que contenga el método main. Este método debe:
 • crear un objeto Empleado con los valores: "Laura", 1500, 35
 • crear un objeto Empleado con los valores: "Carlos", -800, 50
 • mostrar por pantalla sus datos
